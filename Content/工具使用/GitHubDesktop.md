# GitHub Desktop 使用手册

用于记录GitHub Desktop的使用技巧。

## 快捷键

| 快捷键           | 含义                           | 前置条件 |
| ---------------- | ------------------------------ | -------- |
| Ctrl + T         | 打开仓库列表                   |          |
| Ctrl + B         | 显示分支列表                   |          |
| Ctrl + 1         | 显示更改                       |          |
| Ctrl + 2         | 显示历史                       |          |
| Ctrl + Shift + G | 在浏览器中打开GitHub/Gitee仓库 |          |
| Ctrl + Shift + T | 拉取仓库代码                   |          |
|                  |                                |          |
