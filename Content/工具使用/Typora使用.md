## 常用快捷键

| 快捷输入                                   | 含义             | 博客园MD支持              |
| ------------------------------------------ | ---------------- | ------------------------- |
| `[TOC] 回车`                               | 生成目录         |                           |
| `==内容==` 或 `<mark>内容</mark>`          | ==高亮==         |                           |
| `CTRL + SHIFT + `                          | 插入代码行       |                           |
| `CTRL + SHIFT + K`                         | 插入代码块       |                           |
| `CTRL + 回车`                              | 表格插入行       |                           |
| `CTRL + SHIFT +[`                          | 有序列表         |                           |
| `CTRL + SHIFT +]`                          | 无序列表         |                           |
| `CTRL + K`                                 | 插入超链接       |                           |
| `CTRL + SHIFT +I`                          | 插入图片         | 支持`![]()`，不支持 `img` |
| `$\textcolor{red}{文本内容}$`              | 插入带颜色的文本 | 不支持                    |
| `<span style="color:red;">文本内容</span>` | 插入带颜色的文本 | 支持                      |
| `ALT+SHIFT+5`                              | ~~删除线~~       |                           |
| SHIFT + TAB                                | 减少缩进         |                           |
| TAB                                        | 增加缩进         |                           |
| `Win+ :` 或  `:表情名:`<br>CTRL + SHIF + B | 添加表情         | `:smile:` :smile:<br />👌  |
| `CTRL + SHIFT + Q`                         | 插入引用         | > 引用的内容》            |
