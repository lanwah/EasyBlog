### 常用快捷键

| 快捷键         | 功能说明 | 前置条件     |
| -------------- | -------- | ------------ |
| `CTRL+C`       | 复制     | 编辑器有焦点 |
| `CTRL+X`       | 剪切     | 编辑器有焦点 |
| `CTRL+W`       | 扩大选择 | 编辑器有焦点 |
| `CTRL+SHIFT+W` | 减少选择 | 编辑器有焦点 |
|                |          |              |
