## 提示信息

介绍提示信息的输入和显示样式。

### 重要信息

markdown输入：

```markdown
> [!IMPORTANT]
> 重要的提示内容，格式如下面的图片
```

## 颜色文字

### 蓝色粗体文字

`<span style="color:blue;font-weight:bold;">内容</span>`

**效果**

<span style="color:blue;font-weight:bold;">请在这写入内容，这为蓝色文字的效果</span>

### 高亮文字

`<mark>内容</mark>`

**效果**

<mark>高亮文字内容</mark>
